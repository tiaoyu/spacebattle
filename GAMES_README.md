# 太空射击游戏

这是一个使用 Go 和 Ebitengine 游戏引擎开发的太空射击游戏，支持多语言显示。

## 游戏介绍

### 太空射击游戏 (Space Shooter Game)
- **目标**: 消灭敌机获得分数，避免被敌机撞击
- **控制**: 方向键移动飞船，空格键射击
- **特色**: 爆炸效果，敌机AI，生命系统，多语言支持

## 控制说明

### 通用控制
- **ESC键**: 返回主菜单
- **R键**: 重新开始当前游戏（在游戏结束时）
- **L键**: 切换语言（中文/英文）

### 主菜单
- **方向键**: 选择选项
- **回车键/空格键**: 确认选择
- **ESC键**: 退出游戏
- **L键**: 切换语言

## 运行方法

### 方法1: 使用主程序
```bash
go run cmd/game/main.go
```

### 方法2: 使用测试程序
```bash
go run test_games.go
```

### 方法3: 编译后运行
```bash
go build -o game cmd/game/main.go
./game
```

## 技术特性

- **游戏引擎**: Ebitengine v2.8.8
- **编程语言**: Go 1.24.7
- **架构**: 场景管理系统，支持游戏间切换
- **输入处理**: 统一的输入管理系统
- **渲染**: 2D图形渲染，支持颜色和形状绘制
- **多语言支持**: 内置中英文语言包，支持运行时切换
- **字体系统**: 支持中文显示的字体渲染系统

## 开发说明

每个游戏都是独立的场景，实现了 `Scene` 接口：
- `Update()`: 更新游戏逻辑
- `Draw()`: 绘制游戏画面

游戏支持：
- 实时输入处理
- 碰撞检测
- 分数系统
- 游戏状态管理
- 场景切换

## 多语言功能

### 支持的语言
- **中文 (zh)**: 默认语言
- **英文 (en)**: 完整翻译
- **俄文 (ru)**: 完整翻译

### 语言切换
- 在主菜单按 **L键** 可以循环切换语言（中文→英文→俄文→中文）
- 语言切换会立即生效，无需重启游戏
- 当前语言状态会显示在屏幕底部

### 添加新语言
1. 在 `assets/i18n/` 目录下创建新的语言包文件（如 `fr.json`）
2. 在 `internal/i18n/i18n.go` 中添加新语言常量
3. 在 `getBuiltinTexts()` 函数中添加新语言的文本映射

### 自定义语言包
游戏会优先加载 `assets/i18n/` 目录下的JSON语言包文件，如果文件不存在则使用内置语言包。
